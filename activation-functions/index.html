Activation Functions:

Sigmoid

Formula: σ(x) = 1 / (1 + e-x)

Range: (0, 1)

Used for: Binary classification, output layers

ReLU (Rectified Linear Unit)

Formula: f(x) = max(0, x)

Range: [0, ∞)

Used for: Hidden layers, most common activation function

Leaky ReLU

Formula: f(x) = max(αx, x), where α is a small constant

Range: (-∞, ∞)

Used for: Hidden layers, solves "dying ReLU" problem

Tanh (Hyperbolic Tangent)

Formula: tanh(x) = (ex - e-x) / (ex + e-x)

Range: (-1, 1)

Used for: Hidden layers, when zero-centered outputs are needed

Softmax

Formula: softmax(z)i = ezi / Σ ezj

Range: (0, 1) for each output, with sum = 1

Used for: Multi-class classification output layers
